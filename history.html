The History of JavaScript

JavaScript was created in 1995 by Brendan Eich while he was working at Netscape Communications. It was originally developed in just 10 days and first appeared in the Netscape Navigator browser.

The language was initially called Mocha, then changed to LiveScript, and finally named JavaScript to attract Java developers.

In 1997, JavaScript was standardized under the name ECMAScript by ECMA International. Since then, it has evolved through several versions, adding features like classes, modules, and arrow functions.

Today, JavaScript powers the web — used for front-end frameworks like React, Angular, and Vue, and even on the server with Node.js.
