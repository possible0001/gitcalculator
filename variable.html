JavaScript Variables

Variables are containers for storing data values in JavaScript. They allow developers to store, reuse, and change information in a program.

In JavaScript, there are three main ways to declare a variable:

  • var – the old way to declare a variable (function-scoped).
  • let – a modern way (block-scoped, can be updated but not re-declared).
  • const – used for values that should not change (block-scoped and constant).

Example: 

var name = "Abdulbaset";
let age = 18;
const country = "Nigeria";

Here, name, age, and country are all variables holding different data types.
